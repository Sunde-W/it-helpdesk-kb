# How to Fix a Printer That Won't Print  
*If your printer isn’t responding, follow these steps:*  

1. Check that the printer is powered on and connected properly.  
2. Open the print queue and clear any stuck jobs.  
3. Restart the Print Spooler service (Windows: run `services.msc` and restart "Print Spooler").  
4. Reinstall or update the printer driver from the manufacturer’s website.  
5. If issues persist, try connecting the printer to a different USB port or network.
