# How to Reset an Outlook Profile  
*Use this when Outlook is not opening or acting strangely.*  
  
1. Close Outlook completely.  
2. Open Control Panel and select **Mail**.  
3. Click **Show Profiles**.  
4. Click **Add** to create a new profile. Give it a name.  
5. Enter your email account details and finish setup.  
6. Under "When starting Microsoft Outlook, use this profile," choose **Prompt for a profile to be used**.  
7. Launch Outlook and select the new profile.  
8. Once you confirm it works, go back and set **Always use this profile** to the new one.  
9. Delete the old corrupted profile if desired.
