# How to Reset a Windows Password

## Method 1: Using Ctrl + Alt + Del
1. Press **Ctrl + Alt + Del**.
2. Select **Change a password**.
3. Enter old password, then new password twice.
4. Press **Enter**.

---

## Method 2: Via Local Users and Groups
1. Press **Windows + R**, type `lusrmgr.msc`, press **Enter**.
2. Navigate: **Users** → right-click user → **Set Password**.
3. Enter new password.

---

✅ Tip: On a domain, contact your system administrator.
