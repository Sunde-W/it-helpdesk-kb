# Outlook Won't Open  
*Follow these troubleshooting steps to resolve Outlook not opening:*  

1. Ensure Outlook is not already running in the background (check Task Manager and end any Outlook.exe processes).  
2. Restart your computer and try opening Outlook again.  
3. Start Outlook in Safe Mode by running `outlook.exe /safe` from the Run dialog; if it opens, disable add-ins.  
4. Repair the Outlook data file (.pst/.ost) using the Inbox Repair Tool (scanpst.exe).  
5. If issues persist, reinstall or repair Microsoft Office via Control Panel or Settings.
